Recent Projects

Remix

Remix is a browser-based compiler and IDE that enables users to build Ethereum contracts with Solidity language and to debug transactions. We're proud crafters of ZoKrates Remix plugin.

ZoKrates

ZoKrates is a toolbox for zkSNARKs on Ethereum. It includes an easy-to-use domain-specific language for developers to leverage the power of zero-knowledge proofs in their decentralized applications.

solc.js

JavaScript bindings for the Solidity compiler (contract-oriented programming language). Smaller and faster alternative to solc for browser-only environments.

source-verify

Solidity re-compiler used to verify that Ethereum contract bytecode corresponds to certain source code.

ink!

ink! is a Rust-based embedded domain specific language (eDSL) for writing Wasm smart contracts specifically for Substrate-powered blockchains. We're proud crafters of ink! Remix plugin.

Factory Work Optimization

Private Ethereum network implementation that runs on Raspberry PI devices with interface for factory workers.
